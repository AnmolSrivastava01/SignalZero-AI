We are currently migrating the entire 2024 2025 technical manual library from legacy Microsoft OneNote sections into a high-performance, local RAG pipeline. The goal is to allow the on-site engineering team to query hardware specifications and safety protocols without needing an active internet connection or risking IP leakage to OpenAI/Anthropic.

Legacy Server: 192.168.1.45 (Internal Dev Rack   Port 8080).

2. Technical Implementation Details
The Vector Store: We are deploying a local ChromaDB instance running on a Docker container.

Embedding Model: Currently testing BGE-Small-v1.5 to keep latency low on the engineering laptops, but the accuracy on Table of Contents (ToC) mapping is lagging.

Current Logic Gap: The RAG system is failing to resolve cross-references between the "Safety Override" chapter and the "Electrical Grounding" diagram. This is a critical failure because if a tech queries "emergency shutdown," the system currently misses the grounding prerequisite.
